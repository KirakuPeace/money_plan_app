# マネープランアプリ

スマホ・タブレットでも利用できることを想定した、マネープラン入力・試算アプリです。

## 主な機能

- 現況、収入、支出を入力できます
- マネープラン表を表示できます
- 入力データをJSONファイルとして保存できます
- 保存したJSONファイルを読み込んで再開できます

## 使い方

1. 画面に沿って入力します。
2. 「MP表」タブで試算結果を確認します。
3. 「保存」タブで入力データをJSONファイルとして保存します。
4. 次回は「保存」タブでJSONファイルを読み込んで再開します。

## 起動方法

```bash
streamlit run app.py
